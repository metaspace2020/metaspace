# Sharing Annotations & Datasets

## What it is

The annotation page includes a set of share buttons that generate direct
links to a specific annotation, a filtered set of annotations, or an entire
dataset. These links can be sent to collaborators for instant access without
requiring them to navigate through the interface themselves.

## When to use it

When you want to share a specific annotation, a set of annotations, or an
entire dataset with collaborators or reviewers.

## What results look like

You receive a URL that can be copied and shared directly. Recipients with
access to the dataset can open the link and land directly on the shared
annotation or view.
