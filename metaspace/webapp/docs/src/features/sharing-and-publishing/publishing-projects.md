# Publishing Projects

## What it is

Publishing a project makes its datasets accessible outside your private
workspace. METASPACE supports two publishing modes: a private reviewer link
and full public release. Both are managed through a simple form on the
project page.

## When to use it

There are two scenarios for publishing a project:

- **During peer review:** Generate a private link to share with journal
  reviewers. The project remains private and is only accessible to those
  with the link. Access can be revoked after the review process.
- **After acceptance:** Publicly publish the project so that it appears in
  the METASPACE knowledgebase and can be cited in the data availability
  statement of your paper.

## What results look like

For a reviewer link, you receive a private URL. Only recipients with the
link can access the project in read-only mode — it does not appear publicly.

Once publicly published, the project becomes browsable in the METASPACE
knowledgebase and is accessible to anyone.
