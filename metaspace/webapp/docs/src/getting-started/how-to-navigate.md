# How to Navigate the Docs

How you use these docs depends on where you are in your work.

**If you're new to METASPACE**, start with the [Your METASPACE Journey](/getting-started/typical-workflows) page in Getting Started. It maps the most common paths through the platform — from uploading a dataset to exploring annotations and running downstream analyses — and links to the relevant feature pages at each step.

**If you're looking for a specific feature**, go directly to the Features section and find it by domain. Each feature page answers four questions: what the feature is, when to use it, how to use it, and what the results look like.

**If you're trying to interpret something in your results**, go to the Guides section. The scientific interpretation guides cover the [annotation page diagnostics](/guides/interpreting-results/understanding-annotation-page), [co-localization](/guides/interpreting-results/colocalization), [off-sample classification](/guides/interpreting-results/off-sample-filtering), differential analysis, and spatial segmentation results.

**If you're preparing data for upload**, the Before Submission guides cover the [imzML export format](/guides/before-submission/exporting-to-imzml), [metadata recommendations](/guides/before-submission/metadata-recommendations), and how to choose annotation parameters.
