# Account Settings

Your account settings are accessible by clicking your username in the
top-right corner and selecting **My account**. This page is divided into
four sections.

## User Details

Update your **full name**, **email address**, and **password** from this
section. These are the details associated with your METASPACE account and
visible to collaborators you share data with.

## Groups and Projects

The **My account** page provides a quick overview of all the [groups](/getting-started/dataset-organization/groups-and-members) and
[projects](/getting-started/dataset-organization/projects) you belong to, along with your role in each (e.g., Group admin,
Project manager) and the number of datasets associated with each entry.
From here you can also leave a group or project, or create a new project
directly.

For a full view of your groups and projects, use the **My groups** and
**My projects** entries in the same dropdown menu.

## API Access

METASPACE provides an API key for programmatic access and integration with
third-party tools such as the [Python client](/features/tools-and-integrations/python-client).
Using an API key avoids sharing your password with external applications.

From the **API access** section you can:
- **Copy** your existing API key
- **Revoke** the current key and generate a new one if it has been compromised or you want to rotate it
