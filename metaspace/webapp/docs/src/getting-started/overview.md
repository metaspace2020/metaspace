# Overview of the Documentation

The METASPACE docs are organized into three sections:

- **Getting Started** — orientation, navigation, and a stage-by-stage workflow map.
- **Features** — what each feature is, when to use it, and how to use it.
- **Guides** — in-depth references for preparing your data and interpreting your results.

Feature pages tell you how to operate METASPACE. Guides tell you how to understand what it produces. The distinction matters when you're not sure where to look.

Start with [Your METASPACE Journey](/getting-started/typical-workflows) page — a stage-by-stage map from raw data to published results, with links to the right page at each step.
